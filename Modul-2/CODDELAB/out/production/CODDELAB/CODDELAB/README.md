# 🎮 CODDELAB Assignments: Anime Edition

Welcome to the Code Lab! Your mission is to complete the Java programs by filling in the missing code. 
Follow the instructions below to help the Anime characters complete their journeys.

---

## 🤞 Mission 1: Jujutsu Kaisen - ArrayList
**File:** `ArrayListExercise.java`

**Objective:**
Help **Gojo Satoru** manage the student roster for Tokyo Jujutsu High using an `ArrayList`. The list must be dynamic to handle new students, transfers, and... unfortunate incidents.

**Your Tasks:**
1.  **Declaration**: Initialize the `ArrayList` to store the names of sorcerers.
2.  ** recruitment**: Use `.add()` to recruit Itadori, Fushiguro, and Kugisaki.
3.  **Transfer Student**: Use `.add(index, element)` to bring in Yuta Okkotsu.
4.  **Identification**: Use `.get(index)` to check who is currently at a specific rank.
5.  **Possession**: Use `.set(index, element)` to update Itadori's status when Sukuna takes over.
6.  **Casualties**: Use `.remove(index)` to update the roster after the Shibuya Incident.
7.  **Status Report**: Use `.size()` and `.isEmpty()` to check the strength of the school.
8.  **Total Wipeout**: Use `.clear()` for the Culling Game scenario.

## 🏴‍☠️ Mission 2: One Piece - LinkedList
**File:** `LinkedListExercise.java`

**Objective:**
Help **Nami** navigate the Grand Line! You must build a `LinkedList` that represents the chain of islands the Straw Hat crew visits. If the Log Pose link is broken, they will be lost at sea!

**Your Tasks:**
1.  **Map Structure**: Complete the `Island` class (Node) constructor.
2.  **Chart Course**: Implement `addIsland()` to link a new island to the end of the chain.
3.  **Buster Call**: Implement `busterCall()` to delete an island from the map (re-link the previous island to the next one).
4.  **Logbook**: Implement the loop in `printLogbook()` to traverse and print the route.
5.  **Navigation Check**: Implement `isIslandOnRoute()` to search if a specific island exists in the chain.
6.  **Adventure Count**: Implement `countIslands()` to tally up the journey.
